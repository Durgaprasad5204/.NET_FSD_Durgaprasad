# ShopEZ - E-Commerce Frontend Application

A simple e-commerce frontend prototype built with HTML, CSS, JavaScript, jQuery, and Bootstrap.

## Features
- Browse products
- View product details
- Add/remove items from cart
- Checkout simulation
- Cart data saved in browser
- Responsive design

## Pages
- **Home** - Featured products and categories
- **Products** - All products with search and filter
- **Product Details** - Detailed product information
- **Cart** - View and manage cart items
- **Checkout** - Place order (simulated)

## Technologies Used
- HTML5
- CSS3
- JavaScript (ES6)
- jQuery 3.6.0
- Bootstrap 5.3.0
- LocalStorage
- JSON

## How to Run
1. Download the zip file or clone my it from my repository
2. Open `index.html` in any web browser
3. No server or internet connection required
